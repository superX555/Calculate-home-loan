# 計算房貸
這是一款基於 C# WinForms 開發的輕量化房貸試算工具。專為有購屋規劃的需求者設計，提供直觀的介面，幫助使用者精確掌握各階段的還款支出與利息細節。

✨ 核心特色
靈活參數設定：支援自備款、年利率、貸款年限及寬限期的整合計算。

精準還款報表：自動生成包含「寬限期月繳」與「還款期月繳」的專業對照表。

介面優化設計：採用 TableLayoutPanel 佈局，支援高解析度字體與自適應對齊，閱讀不費力。

即時防呆機制：自動檢測自備款金額及寬限期合理性，確保輸入邏輯正確。

📊 試算功能說明
本工具採用業界標準的「本息平均攤還法」進行運算：

房屋總價：輸入預計購買的房屋總金額。

寬限期設定：支援 0 至數年的寬限期，期間僅需繳納利息。

利息支出分析：詳細列出總支付利息與總還款金額，助您評估財務壓力。

🚀 快速開始
環境需求：需安裝 .NET Framework 4.7.2 或更高版本。

執行方式：

使用 Visual Studio 開啟專案檔案 (.sln)。

按下 F5 進行編譯與執行。

使用步驟：輸入貸款參數後，點擊「計算房貸結果」按鈕，下方文字方塊即會噴出完整的報表。

🛠 程式技術棧
開發語言：C#

框架：Windows Forms (WinForms)

佈局容器：TableLayoutPanel

主要類別：StringBuilder (用於動態報表生成), NumericUpDown (提升輸入準確度)
